# Flourish App - Video Documentation Index

This directory contains all documentation needed to create a professional product demo video for Flourish.

## Documents Overview

### 1. VIDEO_SCRIPT.md
**Purpose**: Complete video script with narration, timing, and scene descriptions.

**Contents**:
- Full script with narrator lines
- Scene-by-scene breakdown
- Timing guidelines (3-5 minutes total)
- Production notes
- Script variations (short, extended, social media clips)

**Use When**: Recording the video, preparing voiceover, or reviewing content flow.

---

### 2. VIDEO_SHOT_LIST.md
**Purpose**: Detailed shot-by-shot breakdown with visual specifications and technical details.

**Contents**:
- Every shot specified with timing
- Visual elements for each shot
- Actions and camera movements
- Overlay specifications
- Technical recording settings
- Post-production checklist
- Alternative shots and backup plans

**Use When**: Setting up recording equipment, planning camera work, or during post-production editing.

---

### 3. VIDEO_PRODUCTION_CHECKLIST.md
**Purpose**: Comprehensive checklist covering all aspects of video production from pre-production to publication.

**Contents**:
- Pre-production tasks
- Recording session checklists
- Post-production workflow
- Quality assurance steps
- Publication checklist
- Timeline estimates
- Resource requirements

**Use When**: Managing the entire production process, ensuring nothing is missed, or tracking progress.

---

### 4. VIDEO_SCRIPT_BRIEF.plan.md (Original Brief)
**Purpose**: High-level brief outlining the video concept, features to showcase, and key talking points.

**Contents**:
- Product overview
- Target audience
- Value propositions
- Feature highlights
- Demo flow recommendation
- Key metrics

**Use When**: Understanding the overall vision, presenting to stakeholders, or planning the video concept.

---

## Quick Start Guide

### For Video Producers/Directors:
1. Start with `VIDEO_SCRIPT.md` to understand the narrative flow
2. Review `VIDEO_SHOT_LIST.md` for visual specifications
3. Use `VIDEO_PRODUCTION_CHECKLIST.md` to manage the production process

### For Voiceover Artists:
1. Read `VIDEO_SCRIPT.md` for complete narration
2. Review timing guidelines in each scene
3. Practice with the script before recording

### For Video Editors:
1. Use `VIDEO_SHOT_LIST.md` as editing guide
2. Follow post-production checklist in `VIDEO_PRODUCTION_CHECKLIST.md`
3. Reference `VIDEO_SCRIPT.md` for timing and content

### For Project Managers:
1. Review `VIDEO_PRODUCTION_CHECKLIST.md` for timeline and resources
2. Use `VIDEO_SCRIPT_BRIEF.plan.md` to understand scope
3. Track progress using the checklist

## Video Overview

**Title**: Flourish App - AI-Powered Retail Property Intelligence Demo

**Length**: 3-5 minutes (recommended: 4.5 minutes)

**Format**: Screen recording with voiceover narration

**Target Audience**: Retail property managers, shopping centre leasing teams, property investment analysts

**Key Message**: Flourish helps property managers identify tenant mix gaps, discover revenue opportunities, and automate tenant acquisition using AI-powered analysis.

---

## Key Features Showcased

1. **Dashboard** - Command center with live metrics
2. **Gap Analysis** - AI-powered tenant mix analysis with interactive maps
3. **Gap Fulfillment** - Transform gaps into actionable opportunities
4. **Automated Outreach** - AI finds and engages potential tenants
5. **Reports** - Professional, data-rich reports generated automatically
6. **Voice Assistant** - Natural language property intelligence

---
