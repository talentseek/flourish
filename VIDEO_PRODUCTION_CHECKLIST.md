# Flourish App - Video Production Checklist

## Pre-Production

### Planning
- [ ] Review video script (`VIDEO_SCRIPT.md`)
- [ ] Review shot list (`VIDEO_SHOT_LIST.md`)
- [ ] Identify all locations/scenarios to feature
- [ ] Prepare backup locations in case primary doesn't work
- [ ] Test all UI interactions before recording
- [ ] Verify data is available for featured locations

### Technical Setup
- [ ] Set up screen recording software (Loom, OBS, or similar)
- [ ] Configure recording settings:
  - [ ] Resolution: 1920x1080 minimum
  - [ ] Frame rate: 30fps minimum
  - [ ] Audio: High-quality microphone connected
- [ ] Test recording quality
- [ ] Set up browser:
  - [ ] Use Chrome or Firefox (latest)
  - [ ] Window size: 1920x1080
  - [ ] Zoom: 100%
  - [ ] Disable extensions
  - [ ] Use light theme
- [ ] Prepare cursor settings (large, visible)
- [ ] Clear browser cache and cookies
- [ ] Log in to Flourish app
- [ ] Test internet connection stability

### Content Preparation
- [ ] Verify Queensgate Shopping Centre data is complete
- [ ] Test gap analysis functionality
- [ ] Test gap fulfillment workflow
- [ ] Test outreach functionality
- [ ] Verify reports are available
- [ ] Prepare example questions for voice assistant
- [ ] Have backup location ready (Meadowhall, Bluewater, etc.)

### Script Preparation
- [ ] Review and memorize key talking points
- [ ] Practice voiceover timing
- [ ] Prepare for ad-libs if UI behaves differently
- [ ] Note any script changes needed based on actual UI
