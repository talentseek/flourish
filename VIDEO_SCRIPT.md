## Production Notes

### Screen Recording Guidelines

1. **Resolution**: Record at 1920x1080 minimum
2. **Frame Rate**: 30fps or higher
3. **Cursor**: Show cursor movements clearly, use smooth movements
4. **Transitions**: Use smooth transitions between sections (fade or slide)
5. **Zoom**: Zoom in on important UI elements when explaining features
6. **Pacing**: Allow 2-3 seconds for UI to load before interacting

### Visual Overlays

- Add text overlays for key metrics (e.g., "2,600+ Locations", "£2.4M Revenue Opportunity")
- Use subtle animations for emphasis (e.g., pulse on important buttons)
- Highlight specific UI elements with subtle glow or border when mentioned

### Audio Guidelines

- **Voiceover**: Clear, professional, enthusiastic but not over-the-top
- **Background Music**: Subtle, instrumental, non-distracting (optional)
- **Sound Effects**: Minimal—only for button clicks or transitions (optional)

### Specific UI Interactions to Capture

1. **Dashboard**: 
   - Hover over metric cards
   - Click "Gap Analysis" in sidebar

2. **Gap Analysis**:
   - Type "Queensgate" in search box
   - Select from dropdown
   - Adjust distance slider
   - Toggle location type filters
   - Click "Run Gap Analysis"
   - Scroll through location details
   - Show gap priority cards
   - Show missing brands section

3. **Gap Fulfillment**:
   - Select Queensgate from dropdown
   - Show gap cards
   - Click "Local Outreach" button

4. **Outreach**:
   - Scroll through place cards
   - Select multiple tenants
   - Click "Start Outreach"
   - Show AI-generated messages

5. **Reports**:
   - Scroll through report cards
   - Show key metrics
   - Show report sections

6. **Assistant**:
   - Show example questions
   - Show features list

### Data to Use

- **Primary Location**: Queensgate Shopping Centre, Peterborough, PE1 1NT
- **Example Gaps**: Fast Casual Dining (High), Entertainment & Leisure (Medium), Health & Wellness (Medium)
- **Example Metrics**: 9.5M footfall, 94 retailers, £2.4M revenue opportunity, 4.1★ rating
- **Example Competitors**: Use actual nearby locations from database

### Timing Breakdown

- Opening Hook: 30 seconds
- Dashboard: 30 seconds
- Gap Analysis: 60 seconds
- Gap Fulfillment: 45 seconds
- Automated Outreach: 30 seconds
- Reports: 30 seconds
- Voice Assistant: 15 seconds
- Closing: 30 seconds
- **Total: ~4.5 minutes**

### Post-Production

1. Add intro/outro graphics with Flourish branding
2. Include captions/subtitles for accessibility
3. Add chapter markers for easy navigation
4. Create thumbnail with compelling image and text overlay
5. Export in multiple formats (MP4 for web, optimized version for social media)
